一、替换方法
------------------------------------------------------------
1. 先关闭当前 Electron 程序和 Python 后端。
2. 解压本替换包。
3. 将解压后的文件夹内容整体复制到项目根目录：
   C:\Users\25838\Desktop\Jiaotong-gpt
4. Windows 提示同名文件时，选择“替换目标中的文件”。
5. 重新启动项目。

二、本次修改效果
------------------------------------------------------------
1. normal.onnx 道路异常模式的主视频：
   - 不再绘制四点 ROI；
   - 不再绘制候选车道线；
   - 不再绘制粗车道线；
   - 不再绘制普通车辆框；
   - 只保留异常候选框和已确认异常框，主画面更清楚。

2. 主视频下方新增“道路辅助视图”：
   - 左侧：复用同一视频帧，单独绘制 ROI、候选线段和稳定车道线；
   - 右侧：保留 multi_reference_road_anomaly_demo.py 的蓝/红差分图逻辑；
     蓝色为低差异道路区域/车辆忽略区域，红色为异常投票区域。

3. 没有额外打开第二条视频流：
   - 左侧车道线画面直接复用主视频 DOM 图像；
   - 不会因为辅助视图再次连接手机或再次打开本地视频。

四、验证
------------------------------------------------------------
在项目根目录运行：

python verify_road_visual_split.py

看到“验证通过”后再启动 Electron。

五、使用提示
------------------------------------------------------------
- 选择 normal.onnx。
- 选择四点道路 ROI。
- 正常基准采集过程中，右侧辅助图会先显示蓝色道路有效区。
- 基准完成后，右侧开始显示实时差分；异常投票像素显示红色。
- 主视频只重点显示异常物体，不再被车道线遮挡。
